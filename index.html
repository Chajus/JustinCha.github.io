Nashville Housing Data Cleaning

Tools: SQL, Tableau

Cleaned and transformed raw Nashville housing transaction data using SQL. Tasks included handling NULL values, splitting address fields, standardizing categorical values, and removing duplicate records using window functions. The cleaned dataset was then used to create a Tableau dashboard for visualization.

Key Insights

  • Property sales increased significantly after 2014
  • Certain areas show higher average housing prices
  • Vacant property sales represent a small percentage of transactions

Online Retail Data Cleaning

Tools: SQL, Tableau

Analyzed an online retail dataset using SQL to calculate revenue metrics, identify top-performing products, and analyze customer spending behavior. The cleaned dataset was visualized in Tableau through a dashboard highlighting daily revenue trends and key sales insights.

Key Insights

  • Revenue fluctuated significantly during December, with several spikes indicating higher purchasing activity on certain days.
  • A small number of products generated a large portion of total revenue, highlighting the importance of top-performing items.
  • Customer spending was highly concentrated, with a few customers accounting for significantly higher purchases than others.
